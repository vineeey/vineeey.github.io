# VINAY K P - Futuristic 3D Portfolio

Premium cinematic portfolio built with Next.js App Router, TypeScript, Tailwind CSS, Framer Motion, GSAP, React Three Fiber, and Lenis.

## Scripts

```bash
npm install
npm run dev
npm run lint
npm run build
```

## GitHub Pages Export

This project is configured for static export via `next.config.js`:

- `output: 'export'`
- `images.unoptimized: true`
- `trailingSlash: true`

After build, deploy the generated `out/` output with GitHub Pages workflows/hosting.
